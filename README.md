# IP2Location.io OpenAPI Definitions

IP2Location.io provides [various APIs](https://www.ip2location.io/documentation) to help you to lookup the geolocation information for IP address according to your needs.

This repository contains IP2Location.io API definition files, represented in the OpenAPI Specification standard.

## Folder structure

API definitions in this repository are organized into two sub-folders:

```
   /json – Definition files in the json format.
   /yaml – Definition files in the yaml format.
```

## Usage

Typically, you can import and use the API definition files in the [Postman](https://www.postman.com/). Postman allows you to quickly set up an API request by importing an existing definition files. You can follow their tutorial (https://learning.postman.com/docs/getting-started/importing-and-exporting/importing-from-swagger/) to learn more on how to do it.
